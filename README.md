
# 🚗 Vehicle CO2 Emission Prediction App

## 📌 Project Overview
This project focuses on predicting **vehicle CO2 emissions** using machine learning. It combines **data analysis, feature engineering, and model deployment** into a complete end-to-end pipeline.

The final product is an interactive **Streamlit web application** where users can input vehicle details and get emission predictions in real-time.

---

## 🎯 Objectives
- Perform **Exploratory Data Analysis (EDA)** to understand emission patterns
- Identify key factors affecting CO2 emissions
- Build a predictive machine learning model
- Deploy the model using Streamlit

---

## 📊 Dataset Features
- Make
- Model
- Vehicle Class
- Transmission
- Engine Size
- Cylinders
- Fuel Consumption
- Model Year
- CO2 Emissions (Target)

---

## 🔍 Exploratory Data Analysis (EDA)
Key insights from analysis:

- 🔥 Engine size has a strong positive relationship with emissions  
- ⛽ Fuel consumption is the most important predictor  
- 🔧 Vehicles with more cylinders emit more CO2  
- 🚙 SUVs and trucks produce higher emissions than compact cars  
- 📈 Presence of outliers with extremely high emissions
